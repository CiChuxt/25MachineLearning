# 医药行业高维数据分析总体报告

## 1. 研究目的

本次分析以医药行业多源数据为基础，围绕不同治疗领域在不同年份中的研发活跃度、临床成功情况、药物审批成果、商业价值和疾病负担进行综合比较。分析的核心目标是把原始的多张表格数据整合为统一的“年份 × 治疗领域”样本，并利用降维和聚类方法识别医药研发领域中的主要结构差异。

本次实际完成的主线为：

1. 构建高维特征表；
2. 对数值变量进行标准化；
3. 使用 PCA 提取主要综合特征；
4. 使用 K-Means 对样本进行聚类；
5. 使用 PCA 二维投影展示聚类结构。

原流程中计划使用 UMAP 做非线性可视化，但当前 MATLAB 环境未安装可用的 `run_umap` 或 `umap` 函数，因此本次结果中的二维可视化采用 PCA fallback，而不将其解释为真正的 UMAP 结果。

## 2. 数据来源与特征构造

本次分析使用当前目录下的 5 个 CSV 文件：

| 文件 | 主要用途 |
|---|---|
| `clinical_trials.csv` | 构造临床试验数量、Phase 3 数量、成功率、失败率、平均入组人数、平均试验周期、平均股价影响 |
| `drug_approvals.csv` | 构造药物审批数量、重磅药数量、超级重磅药数量、峰值销售额、生物药和小分子药审批数量 |
| `disease_burden.csv` | 构造疾病负担指标，包括全球 DALYs 和区域 DALYs |
| `biotech_funding.csv` | 构造年度投融资背景变量，包括交易数量、巨额交易数量、融资总额和平均融资额 |
| `pharma_companies_financials.csv` | 构造年度行业财务背景变量，包括公司数量、总收入、平均经营利润率、总研发支出和管线规模 |

样本单位为：

```text
year × therapy_area
```

最终得到 204 个样本，使用 26 个数值特征。由于各特征单位差异较大，例如试验数量、销售额、DALYs、融资额和研发支出不在同一量纲上，因此在 PCA 和 K-Means 之前统一进行了 Z-score 标准化。

## 3. PCA 结果分析

PCA 的前 8 个主成分累计解释方差如下：

| 主成分 | 单个解释方差 | 累计解释方差 |
|---|---:|---:|
| PC1 | 23.41% | 23.41% |
| PC2 | 16.70% | 40.11% |
| PC3 | 11.95% | 52.06% |
| PC4 | 11.40% | 63.47% |
| PC5 | 8.55% | 72.02% |
| PC6 | 6.42% | 78.44% |
| PC7 | 5.67% | 84.10% |
| PC8 | 3.81% | 87.91% |

可以看出，前 5 个主成分已经解释约 72.02% 的总体信息，前 8 个主成分解释约 87.91%。因此，数据中确实存在较明显的低维结构，PCA 可以较好地压缩原始高维变量。

### PC1：研发活跃度与商业化成果

PC1 的主要正向贡献变量包括：

- `approval_count`
- `small_molecule_approval_count`
- `blockbuster_count`
- `biologic_approval_count`
- `trial_count`
- `phase3_trial_count`

这说明 PC1 主要反映治疗领域在某一年中的研发和商业化活跃程度。PC1 得分越高，通常意味着该领域试验更多、后期试验更多、获批药物更多、重磅药数量更多。因此，PC1 可以解释为“研发与商业化规模”。

### PC2：行业整体规模与研发投入背景

PC2 的主要贡献变量包括：

- `pharma_pipeline_total_est`
- `pharma_company_count`
- `pharma_rd_spend_total_usd_bn`
- `pharma_revenue_total_usd_bn`
- `funding_deal_count`
- `funding_total_usd_bn`

PC2 主要受年度行业背景变量影响，反映整个医药行业在某一年中的规模、收入、研发投入和管线扩张情况。因此，PC2 更接近“行业年度发展水平”。

### PC3：投融资强度与资本活跃度

PC3 的主要贡献变量包括：

- `funding_avg_value_usd_bn`
- `funding_total_usd_bn`
- `funding_megadeal_count`
- `pharma_operating_margin_avg_pct`
- `funding_deal_count`

PC3 主要体现资本市场和投融资活动的强弱，尤其区分融资金额高、巨额交易多的年份。因此，PC3 可解释为“资本活跃度”。

### PC4：临床成功与市场反馈

PC4 的主要贡献变量包括：

- `success_rate`
- `failure_rate`
- `avg_stock_impact_pct`
- `mapped_disease_count`
- `avg_peak_sales_usd_bn_est`

其中成功率和平均股价影响为正，失败率为负，说明 PC4 主要体现临床结果质量和市场反应。因此，PC4 可解释为“临床有效性与市场反馈”。

### PC5：疾病负担与商业价值差异

PC5 的主要贡献变量包括：

- `disease_burden_regional_sum_millions`
- `disease_burden_global_millions`
- `mapped_disease_count`
- `mega_blockbuster_count`
- `avg_peak_sales_usd_bn_est`
- `total_peak_sales_usd_bn_est`

PC5 把疾病负担和商业化价值区分开来，说明高疾病负担领域未必总是对应最高销售额或最多超级重磅药。这一点对分析“医疗需求与研发资源是否匹配”很重要。

## 4. K-Means 聚类结果分析

本次使用 PCA 得分作为聚类输入，并设置 `K = 3`。聚类结果如下：

| 聚类 | 样本数 | 主要特征 |
|---|---:|---|
| Cluster 1 | 99 | 主要覆盖 2010-2018 年，多数治疗领域均衡分布，研发和审批水平中等偏低 |
| Cluster 2 | 86 | 主要覆盖 2019-2026 年，多数治疗领域均衡分布，行业背景变量更高 |
| Cluster 3 | 19 | 主要由 oncology 构成，少量 neurology 和 infectious，研发与审批强度显著最高 |

### Cluster 1：早期稳定发展型

Cluster 1 包含 99 个样本，年份集中在 2010-2018 年。该类样本覆盖大多数治疗领域，包括 metabolic、psychiatry、respiratory、rare_disease、gastrointestinal、dermatology、cardiovascular、infectious、immunology 等。

该类平均临床试验数量为 2.40，平均审批数量为 2.35，平均重磅药数量为 0.55，平均总峰值销售额约为 2.95 十亿美元。整体看，这一类代表相对早期阶段的常规研发与审批活动，行业背景变量也低于后期年份。

### Cluster 2：后期行业扩张型

Cluster 2 包含 86 个样本，年份集中在 2019-2026 年。该类同样覆盖大多数治疗领域，但年度行业背景变量明显高于 Cluster 1。例如，平均融资总额约为 145.85 十亿美元，平均总研发支出约为 163.79 十亿美元，均高于 Cluster 1。

该类平均临床试验数量为 2.08，平均审批数量为 2.16，治疗领域本身的试验和审批强度并不明显高于 Cluster 1，但行业整体资金、收入、研发支出和管线规模更高。因此，Cluster 2 更适合解释为“后期行业扩张型”。

### Cluster 3：高研发高商业化集中型

Cluster 3 包含 19 个样本，主要由 oncology 构成，其中 oncology 占 15 个样本，另有少量 neurology 和 infectious 样本。

该类特征非常突出：

- 平均临床试验数量为 9.58，远高于 Cluster 1 和 Cluster 2；
- 平均 Phase 3 试验数量为 5.74，说明后期研发成熟度更高；
- 平均审批数量为 15.95，远高于其他两类；
- 平均重磅药数量为 3.26；
- 平均总峰值销售额为 17.48 十亿美元；
- 平均疾病负担为 262.89 百万 DALYs；
- 平均成功率为 0.34，低于其他两类；
- 平均股价影响为 -4.90%，说明虽然商业化活跃，但临床和市场风险也更高。

因此，Cluster 3 可以解释为“高研发投入、高审批产出、高商业价值但高风险”的治疗领域集合。其中 oncology 是最典型代表。

## 5. 重点发现

### 5.1 Oncology 是最突出的高活跃领域

在按试验数量、审批数量、重磅药数量和总峰值销售额排序时，oncology 多次位于前列。例如 2024 年 oncology 的试验数量为 12，审批数量为 26，重磅药数量为 6，总峰值销售额约为 20.24 十亿美元。2023 年 oncology 审批数量也达到 26。

这表明肿瘤领域在数据中同时具有高研发活跃度、高审批产出和高商业价值，是医药研发资源高度集中的核心领域。

### 5.2 高疾病负担不一定对应高研发和高审批

疾病负担最高的样本主要是 cardiovascular，例如 2026 年 cardiovascular 的全球疾病负担达到 592 百万 DALYs，但其临床试验数量为 0，审批数量为 1，总峰值销售额仅约 0.34 十亿美元。

这说明疾病负担与研发资源配置之间并非完全匹配。某些疾病负担很高的领域并没有表现出同等水平的研发和商业化活跃度，这可以作为报告中的重要讨论点。

### 5.3 资本和行业背景变量造成明显年份分层

Cluster 1 与 Cluster 2 的治疗领域构成类似，但年份分布明显不同：Cluster 1 集中于 2010-2018 年，Cluster 2 集中于 2019-2026 年。这说明投融资、公司收入、研发支出和管线规模等年度行业背景变量对聚类有较大影响。

因此，如果研究问题更关注“不同治疗领域之间的结构差异”，可以考虑在后续稳健性分析中关闭年度行业背景变量，即在脚本中设置：

```matlab
useIndustryBackground = false;
```

这样可以减少年份宏观变量对治疗领域聚类的影响。

### 5.4 高商业化活跃领域伴随更高临床风险

Cluster 3 虽然在试验数量、审批数量和销售额上显著领先，但平均成功率只有 0.34，平均失败率达到 0.51，平均股价影响为负。这说明高价值治疗领域并不意味着低风险，反而可能因为研发难度高、竞争激烈、临床终点复杂等因素而表现出更高失败率。

这一发现可以用于解释医药研发中的典型现象：资源集中于高价值领域，但高价值领域也往往伴随更高不确定性。

## 7. 方法局限

本次分析仍有几个需要说明的局限：

1. 疾病到治疗领域的映射是根据疾病名称手动归类完成的，存在一定简化。
2. 投融资和公司财务数据是年度级别变量，并非治疗领域级别变量，会增强年份效应。
3. K-Means 需要预设聚类数，本次选择 `K = 3` 主要是为了保证结果清晰、易解释。
4. 当前环境没有真正运行 UMAP，因此非线性结构展示尚未完成。
5. 数据中部分变量为模拟或估计型指标，解释时应强调其分析性而非真实市场预测。

## 8. 总体结论

本次分析表明，医药行业不同治疗领域之间存在明显的结构性差异。PCA 结果显示，影响样本差异的主要因素可以概括为研发与商业化规模、行业年度发展水平、资本活跃度、临床成功与市场反馈、疾病负担与商业价值差异。K-Means 聚类进一步将样本划分为早期稳定发展型、后期行业扩张型和高研发高商业化集中型三类。

其中，oncology 是最典型的高研发、高审批、高商业价值领域，但同时也伴随较低临床成功率和更负面的平均市场反应，体现出高价值与高风险并存的特点。另一方面，cardiovascular 等疾病负担较高的领域并未总是表现出相应水平的研发和审批活跃度，说明医药研发资源与疾病负担之间存在一定结构性错配。

总体而言，本次 PCA + K-Means 分析能够较好地揭示医药研发资源、商业化成果和疾病负担之间的关系，为后续报告中讨论“研发资源是否与医疗需求匹配”提供了量化依据。
